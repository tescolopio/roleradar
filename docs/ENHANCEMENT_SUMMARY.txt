1. VIEW CURRENT CONFIGURATION:
   python config_manager.py show

2. USE DEFAULT (NO CHANGES NEEDED):
   python scheduler.py

3. CUSTOMIZE ROLES (OPTIONAL):
   python config_manager.py set-roles "security engineer, compliance officer, CISO"

4. CUSTOMIZE SCHEDULE (OPTIONAL):
   python config_manager.py set-schedule "08:00, 12:00, 15:00"

5. RUN SCHEDULER:
   python scheduler.py
